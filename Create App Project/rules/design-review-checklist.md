### 3.2 API 엔드포인트 정의

| 기능 | Method | Endpoint | Request | Response |
|------|--------|----------|---------|----------|
| 전체 조회 | GET | `/getAll` | - | `{ items: [...] }` |
| 검색 | GET | `/search?q=` | query | `{ results: [...] }` |
| 상세 조회 | GET | `/getById?id=` | id | `{ item: {...} }` |
| 생성 | POST | `/create` | body | `{ success: true }` |
| 수정 | PUT | `/update` | body | `{ success: true }` |
| 삭제 | DELETE | `/delete?id=` | id | `{ success: true }` |

### 3.3 데이터 스키마 정의

```typescript
// 예시: 메인 데이터 타입
interface DataItem {
  id: string;
  title: string;
  description?: string;
  category?: string;
  createdAt: string;
  updatedAt: string;
}
```
